Curator Privacy Policy

  • What information do you collect?
    • This application will store user's information in that user's Google Drive. It will use OAuth to access that drive for the purpose of storing the user's application data. Otherwise, this application collects no information for any purpose.
  • How do you use the information?
    • This application does not use any information provided by the user except to store that user's application data on that user's Google drive.
  • What information do you share?
    • This application and the creator of this application do not share any information provided by the user.
